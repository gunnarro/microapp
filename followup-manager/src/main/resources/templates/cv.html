Career Summary

Currently I work as System architect at Telenor AS in Norway. Where my main task is further develop and maintain the integration architecture of the television platform. Where my responsibility are both application and platform architecture of our integration platform. The TV platform is mainly using Oracle Service Bus as integration layer between other systems, both internal and external systems. Where we basically use SOAP as communication protocol We are also using REST based microservices running as Spring Boot components in an Openshift platform. In the future, the platform will only run on Openshift as java spring boot applications. I have basically worked in a scrum team with test driven development, all the time. All the time I have worked as well as developer as architect. After finishing my MSc degree in 2000 i started as consultant at Accenture. Where I worked with backend and integration bank business systems using java as programming language.
